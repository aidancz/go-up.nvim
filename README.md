# 🆙 Go-Up.nvim

"Go up" is what you used to have to say when vim wouldn't scroll up past line 1.
Now you don't have to.

## 👆 How to Install

Lazy.nvim config:

```lua
{
    'nullromo/go-up.nvim',
    opts = {}, -- specify options here
    config = function(_, opts)
        local goUp = require('go-up')
        goUp.setup(opts)
    end,
}
```
